Help us with our Covid-19 Effort!

The Butte Lab made the Covid Counties project open source so that the entire scientific community can benefit from and contribute to it. Our code is kept up to date on Github and also Dockerhub. Both sites provide in-depth tutorials on how to get started. The tutorial covers how to code Covid Counties on any device or operating system. We hope this code helps other programmers with their work. We also look forward to future contributions being made to Covid Counties not by ourselves, but by the scientific community. If you have additions to the site or would like to collaborate do not hesitate to reach out!
